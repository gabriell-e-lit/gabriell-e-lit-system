# Обяснение на схемата (кратко и ясно)
1. Трите източника на истина
WordPress → динамика
Static Site → канон
e‑books → пълни текстове

2. Всеки слой подава данни към екстрактора
Категории → Issue/Year/Genre
Постове → автори, книги, изображения
Статичен сайт → канон (автори, книги, изложби)
e‑books → метаданни + текстове

3. Екстракторът прилага приоритети
H2 > H3 > Tag > Post > Static > e‑books

4. Конфликтите се решават в полза на канона
5. Всичко се обединява в DataTypes v2
Author / Book / Artwork / Exhibition

6. Това изгражда цялата платформа
Статични страници, списание, галерия, библиотека, навигация.
